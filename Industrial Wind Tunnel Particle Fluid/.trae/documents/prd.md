
## 1. Product Overview
空气动力学粒子流体模拟工具，用于汽车和飞机设计中的风洞测试可视化，帮助工程师直观理解风阻、涡流和气流分离现象。
- 主要功能：3D模型展示、百万级粒子气流模拟、实时风向控制、翼型参数管理
- 目标用户：空气动力学工程师、汽车/飞机设计师

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| Engineer | 无需注册 | 使用所有功能，保存和加载翼型数据 |

### 2.2 Feature Module
1. **3D场景页面**：模型展示、粒子模拟、风向控制
2. **翼型管理页面**：数据增删改查、参数配置
3. **预设效果展示**：4种预设模拟场景

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 3D场景页 | 3D画布 | 渲染汽车/飞机3D模型 |
| 3D场景页 | 粒子系统 | 数百万粒子的WebGL Shader渲染 |
| 3D场景页 | 风向控制 | 鼠标拖拽改变风向角度，实时重新计算粒子轨迹 |
| 3D场景页 | 预设效果 | 4种预设模拟场景的一键切换 |
| 翼型管理页 | 数据管理 | 翼型数据和雷诺数参数的CRUD |

## 3. Core Process
用户进入3D场景页面，选择汽车或飞机模型，通过鼠标拖拽控制风向，观察粒子流动效果。用户可切换预设场景（流线型、失速涡流、底盘乱流、侧向强风），也可在翼型管理页添加、编辑翼型数据。

```mermaid
graph TD
  A[进入应用] --&gt; B[3D场景页面]
  B --&gt; C[选择模型]
  C --&gt; D[鼠标控制风向]
  B --&gt; E[切换预设效果]
  B --&gt; F[翼型管理页面]
  F --&gt; G[CRUD翼型数据]
```

## 4. User Interface Design

### 4.1 Design Style
- **主要颜色**：深海军蓝 (#0a1628) 背景，青蓝色 (#00d4ff) 主色，橙红色 (#ff6b35) 警示色
- **按钮风格**：简约圆角按钮，带有微妙发光效果
- **字体**：现代无衬线字体，使用 Inter 或 JetBrains Mono
- **布局风格**：左侧控制面板，右侧主画布，深色工业风
- **图标风格**：Lucide React 线性图标，科技感强

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 3D场景页 | 控制面板 | 深蓝色侧边栏，白色文字，带发光边框的输入控件 |
| 3D场景页 | 3D画布 | 深色背景，HDR环境，带后处理效果 |
| 3D场景页 | 预设按钮 | 横向排列，带hover动画和选中状态高亮 |
| 翼型管理页 | 数据表格 | 工业风格表格，带斑马纹和悬停效果 |

### 4.3 Responsiveness
- 桌面端优先，适配1080p及以上分辨率
- 移动端适配为响应式布局，画布自适应屏幕

### 4.4 3D Scene Guidance
- **环境**：深色科技感空间，带微妙HDR环境光
- **光照**：三点照明系统，主光源带蓝色调，强调科技感
- **相机**：初始使用45度俯视视角，支持轨道控制
- **交互**：鼠标拖拽控制风向，鼠标滚轮缩放，右键平移
- **粒子效果**：使用自定义WebGL Shader实现流畅的粒子动画
- **后处理**：Bloom发光效果，提升视觉冲击力
